# Contributing

This repository is the open, canonical home of the **DealerMAX Public API**
specification and examples. The API itself is operated by DealerMAX; the
`openapi/dealer-public-api.json` spec is generated from the production service
and mirrors the live contract.

## Good contributions

- Fixes to example code (cURL, Node, Python) and documentation.
- New examples in additional languages that follow the backend-first, key-stays-server pattern.
- Typo and clarity fixes in this README and the examples.

## Please note

- The OpenAPI file is **generated** from production. Open an issue describing a
  contract problem rather than editing the spec by hand — we regenerate it from source.
- For questions about access, keys, or rate limits, see
  [developers.dealermax.app](https://developers.dealermax.app) or email **support@dealermax.app**.
- Never include a real API key in an issue, PR, or example.
